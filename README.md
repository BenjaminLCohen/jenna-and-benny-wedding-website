# Jenna & Benjamin Wedding Website

A static wedding website for Jenna Silverman and Benjamin Cohen.

## Local Preview

Open `index.html` in a browser. No build step is required.

## Deploy With GitHub and Vercel

1. Create a new GitHub repository.
2. Push this folder to the repository.
3. In Vercel, choose **Add New Project** and import the GitHub repo.
4. Use the **Other** framework preset.
5. Leave the build command empty and deploy from the project root.

## Connect a GoDaddy Domain

1. In Vercel, open the project and go to **Settings > Domains**.
2. Add the GoDaddy domain you want to use.
3. In GoDaddy DNS, add the records Vercel gives you.
4. Wait for DNS verification, then make the Vercel domain the production domain.
